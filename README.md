# HSR Relic Helper

崩壊：スターレイルの遺物管理・逆引きツールです。

## 機能概要

### 1. キャラクター遺物管理
キャラクターごとに推奨される遺物セット、メインステータス、サブステータスを登録・管理できます。
- **キャラクター追加・編集**: キャラクター名、遺物セット（トンネル・オーナメント）、メインステータス、サブステータスを設定できます。

### 2. 遺物逆引き検索
遺物セットから、そのセットを必要とするキャラクターを逆引き検索できます。
- **トンネル遺物 / 次元界オーナメント切り替え**: タブで表示を切り替えられます。
- **詳細表示**: 各セットを展開すると、部位ごとのメインステータスやサブステータスの組み合わせごとにキャラクターがグループ化されて表示されます。
- **ナビゲーション**: キャラクター名をクリックすると、そのキャラクターの編集画面へジャンプします。

### 3. 設定・データ管理
- **データのインポート/エクスポート**: 登録したデータをJSON形式でバックアップ・復元できます。
- **遺物データ管理**: ゲーム内の新しい遺物セットが追加された場合に、手動でセットデータを追加・編集できます。

## データ保存場所
データは以下の場所に保存されます（Windowsの場合）:
`%APPDATA%\hsr-relic-helper\`
- `hsr-relic-helper-data.json`: キャラクター設定データ
- `hsr-relic-helper-relics.json`: 遺物セット定義データ
